listas en html

Para hacer listas en html se usaran las etiquetas -li- que significa lista y -ul- que significa lista sin orden

ej:

lista de prioridades
  • fisica
  • programacion
  • calculo:
    • integral
    • diferencial
    • multivariado
  • papitas
tambien estan las listas ordenadas las cuales manejan la etiqueta -ol- Lista ordenada:
  1. paso 1
  2. paso 2
  3. paso 3
  4. paso 4
